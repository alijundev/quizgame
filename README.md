# 🎮 Quiz Game - Java

Quiz Game adalah aplikasi sederhana berbasis CLI (Command Line Interface) yang ditulis dalam bahasa pemrograman Java. Aplikasi ini akan menguji pengetahuan kamu seputar teknologi, pemrograman, dan jaringan dasar melalui serangkaian soal pilihan ganda. 

> **Catatan:** Project ini dibuat untuk memenuhi tugas **Ujian Tengah Semester (UTS)** mata kuliah **Pemrograman Berorientasi Objek (PBO)**.

Project ini juga merupakan implementasi dari konsep-konsep dasar OOP pada bahasa pemrograman Java.

## ✨ Fitur Project

* **Soal Pilihan Ganda**: Menampilkan pertanyaan dengan opsi jawaban A, B, C, atau D.
* **Validasi Jawaban**: Sistem akan langsung mengecek apakah jawaban yang diberikan benar atau salah.
* **Sistem Skor**: Menghitung secara otomatis jawaban yang benar dan menampilkan skor akhir setelah semua pertanyaan dijawab.

## 🚀 Cara Clone Project

Gunakan perintah Git berikut untuk menyalin source code project ini ke komputer lokal kamu:

```bash
# Clone repository
git clone https://github.com/alijundev/quizgame.git

# Masuk ke dalam direktori project
cd quizgame
```

## 💻 Cara Menjalankan Project

Pastikan komputer kamu sudah terinstal **[Java Development Kit (JDK)](https://www.oracle.com/java/technologies/downloads/)**.

Berikut adalah langkah-langkah untuk menjalankan programnya:

1. Buka Terminal atau Command Prompt.
2. Jalankan program secara langsung menggunakan perintah `java` pada folder project:
   ```bash
   java Main.java
   ```
